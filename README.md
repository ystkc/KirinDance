# KirinDance
 迳口村麒麟舞的视觉识别和评分系统
 当前版本用于创建回档点，不能运行

## 使用方式
1. 将要学习的麒麟舞视频放在根目录下
2. 将视频名称包括扩展名拷贝入主程序的standard_video_path变量中
3. 配置环境，安装例如opencv-python库
4. 运行主程序
